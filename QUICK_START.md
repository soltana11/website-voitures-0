# BENA MOTORS - Accès Admin Rapide

## ⚡ Démarrage Rapide (2 minutes)

### 1️⃣ Ouvrez ce lien:
```
https://votre-domaine.netlify.app/admin/
```

### 2️⃣ Remplissez le formulaire:
- Nom du véhicule
- Année
- Kilométrage
- Prix
- Photo (URL)
- Description

### 3️⃣ Cliquez "Ajouter/Mettre à jour"

✅ **C'est tout!** Votre véhicule est en ligne!

---

## Gestion Rapide

| Action | Comment |
|--------|---------|
| **Ajouter** | Remplir + Cliquer "Ajouter" |
| **Modifier** | Cliquer "Modifier" + Changer + Cliquer "Ajouter" |
| **Supprimer** | Cliquer "Supprimer" + Confirmer |
